Get Books

Relative URL: /getBooks

Supported Methods: GET and HEAD

Query Parameters:

Title: The title of the book. Required

Author: The author of the book. Required

Language: The language the book is written in. Required

Year: The year the book was published. Required

Returns:

A JSON Object containing a response message and an array with a single JSON object containing the book's data

Example:

URL: /getBooks?title=Things%20Fall%20Apart&author=Chinua%20Achebe&language=English&year=1958

Returns: {
message: "Book Found",
books: [
{
"author": "Chinua Achebe",
"country": "Nigeria",
"language": "English",
"link": "https://en.wikipedia.org/wiki/Things_Fall_Apart\n",
"pages": 209,
"title": "Things Fall Apart",
"year": 1958,
"genres": [
"Historical Fiction",
"Postcolonial Literature"
]
}
]
}

Get Author

Relative URL: /getAuthor

Supported Methods: GET and HEAD

Query Parameters:

Author: The author of the books. Required

Returns:

A JSON object containing a response message and an array of JSON objects containing data for all the books with the same author.

Example:

URL: /getAuthor?author=Hans%20Christian%20Andersen

"message": "Author found returning all books",

"Fairy Tales",

]
}
]
}

Add Book

Relative URL: /getYear

Supported Methods:POST

Body Parameters:

Title: The title of the book. Required

Author: The author of the book. Required

Language: The language the book is written in. Required

Year: The year the book was published. Required

Returns:

A JSON object representing the new book added to the database

Example:

URL: /addBook

Body: {title: 'v', author: 'Ben Kingsley', language: 'English', year: '2026'}

Returns: {"author":"Ben Kingsley","language":"English","title":"v","year":2026}

Review Book

Relative URL: /getYear

Supported Methods:POST

Body Parameters:

Title: The title of the book. Required

Score: The new review score Required

Returns:

A JSON object representing the new book added to the database

Example:

URL: /reviewBook

Body: {title: 'v', score: '5'}

Returns: No Content
